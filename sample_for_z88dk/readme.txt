bgm_driver を z88dk な C言語から呼び出すサンプルプログラム
--------------------------------------------------------------------------------

1. ビルドの方法 (Windows環境)
	z88dk の SDK に含まれる「z88dk_prompt.bat」を実行して、コマンドプロンプトを
	起動します。

		xxxx\z88dk>■

	cd して、sample_for_z88dk のフォルダへ移動します。

	build.bat を実行します。

	BGMTEST.COM と BGMDRV.BIN が出来上がるので、これを MSX のディスクドライブへ
	コピーして、MSX-DOS上から BGMTEST を実行します。

	z88dk の他に、python3.x がインストールされている必要があります。

2. 使い方
	config.txt の start_address に、BGMDRV.BIN を配置するアドレスを記入します。
	これは 4000h以上のアドレスを指定する必要があります。
	0000h-3FFFh は、頻繁にスロットが切り替わるため、そこに配置すると割り込み発生時
	に、 bgm_driver を見失って暴走します。

	bgmdriver.h に各関数の説明が書かれています。

	何よりも先に実行しなければならないのは、 bgmdrv_init(); です。
	これは、BGMDRV.BIN を config.txt で指定したアドレスに読み込んで、割り込みフック
	をかける処理を実施します。
	失敗した場合は、0 が返ってきます。
	成功した場合は、1 が返ってきます。
	成功した場合は、プログラム終了前までに bgmdrv_term() を実行する必要があります。

	bgmdrv_term() は、後始末処理です。割り込みフックを元に戻します。
	これを実行した後は、演奏開始などの bgm_driver関数は無効になります。
